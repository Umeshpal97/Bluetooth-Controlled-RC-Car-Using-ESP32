Bluetooth Controlled RC Car using ESP32 and Dabble App

Description

This project demonstrates a Bluetooth controlled RC car using ESP32.
The car is controlled using the Dabble Gamepad mobile application.

Features

Wireless Bluetooth control
PWM based speed control
Forward / Backward / Left / Right movement
Smooth motor driving using L298N motor driver

Hardware Used

ESP32
L298N Motor Driver
4 DC Motors
Car Chassis
Li-ion Battery

Mobile App

Dabble App → Gamepad Module

Bluetooth Name

Umesh_Car

How to Run

Install Arduino IDE and ESP32 board package
Install DabbleESP32 library
Upload the code to ESP32
Connect mobile with Bluetooth device Umesh_Car
Open Dabble → Gamepad → Control the car 🎮
